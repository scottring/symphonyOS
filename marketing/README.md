## üé® Key Messaging Pillars

These themes should be consistent across all materials:

### 1. **Execution Advantage**
*"Ready when it's time"* not just "organized"
- Context attachments that surface when needed
- No searching for phone numbers, links, notes
- Everything attached to the task/event

### 2. **Visual Coordination**
*"See conflicts before chaos"*
- Timeline Companion subway-map view
- Family coordination at a glance
- Convergence/divergence visualization

### 3. **Desktop + Mobile Philosophy**
*"Plan deeply, execute quickly"*
- Desktop for weekly planning sessions
- Mobile for quick capture and execution
- One system, zero friction

### 4. **AI as Assistant**
*"Helps you think, doesn't replace thinking"*
- Meal planning prompt ‚Üí full week plan
- Planning session structuring
- Not autonomous, but collaborative

### 5. **Family-First (But Solo-Friendly)**
*"Your family's operating system"*
- Built for coordination, not just tasks
- Fair assignment visibility
- Solo users welcome too
