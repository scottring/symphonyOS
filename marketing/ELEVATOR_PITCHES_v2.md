## What Symphony Is NOT

- Email client (tasks can reference emails)
- Code editor (tasks reference work, IDE is separate)
- Full budgeting tool (could track balances, but not categorization)
- Replacement for Google Calendar (GCal is source of truth, Symphony enhances it)
- AI assistant (AI features coming, but execution is human-driven)
